## 今日のテーマ

ChatGPTを「文章の助手」として使いこなす。下書き・整理・チェックリストを一気に習得。

## 学ぶこと

### 1. 文章作成（返信下書き・Chatwork投稿文）

ChatGPTは文章の下書き作成が得意です。自分で一から書くのではなく、「下書きを作ってもらってから直す」という使い方が効率的です。

### 2. 箇条書き整理・要約

長い文章やメモを整理させることができます。

```
以下のメモを、分かりやすく箇条書きに整理してください：
[メモ内容]
```

### 3. チェックリストを作らせる

```
あなたは業務改善の専門家です。
以下の業務フローのチェックリストを作成してください：
[業務の概要]
抜け漏れが起きやすいポイントも含めてください。
```

### 4. 抜け漏れ確認

```
以下の[書類名/対応内容]を確認し、抜け漏れがあれば指摘してください：
[確認してもらいたい内容]
```

## 具体的な活用例

### 出荷確認チェックリスト

```
あなたは物流業務に詳しい先輩担当者です。
Next EngineからLogizardへの出荷指示フローのチェックリストを作ってください。
EC受注の出荷処理で抜け漏れが起きやすいポイントも含めてください。
```

### 問い合わせ返信の下書き

```
あなたはECサイトのカスタマーサポートの先輩社員です。
以下のような問い合わせメールが来ました：
「注文した商品がまだ届いていません。どうなっていますか？」

条件：
- 出荷は完了しているが、追跡番号はまだ提供していない
- 配送会社への確認が必要な状況
丁寧な返信下書きを作ってください。
```

## 練習ワーク

1. 自分の業務でよく使うチェックリストを1つChatGPTに作らせる
2. または、よくある問い合わせへの返信下書きを作らせる
3. 出てきた内容を確認し、改善したい点を考える

> ⚠️ **注意：** 個人情報・顧客情報・機密情報は入力しないでください。
