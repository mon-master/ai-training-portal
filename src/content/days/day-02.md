## 今日のテーマ

「短い質問」から「背景つきのお願い」へ。ChatGPTの回答を劇的に変えるコツを身につける。

## 学ぶこと

### お願いの基本型

背景・目的・条件・出力形式の4つを入れると、ChatGPTの回答が大きく変わります。

```
あなたは[役割]です。
私は[自分の立場・状況]です。
[背景・目的]
[条件]
[出力形式]でお願いします。
```

### 具体的な変化

**短い質問（Before）：**
```
メール返信 書いて
```

**背景つきのお願い（After）：**
```
あなたはECサイトのカスタマーサポートに慣れた担当者です。
私はメールワイズで問い合わせ対応をしています。
お客様から「注文した商品が届かない」という問い合わせが来ています。
注文番号は省略しますが、出荷済みで追跡番号をお伝えできます。
丁寧で簡潔な返信文の下書きを作ってください。
```

## 業務別：お願いの書き方

### Chatwork報告文

```
あなたは社内向けの連絡文を書き慣れた先輩社員です。
私は[部署名]担当です。
[状況の説明]
[理由・原因]
Chatworkで[宛先]に報告する短い文を作ってください。
```

### 日報・週報

```
あなたは業務日報の作成を手伝うアシスタントです。
私は[担当業務]をしています。
今日の作業メモ：[メモの内容]
これをサイボウズオフィスの日報形式で整理してください。
箇条書きで、各項目2〜3行以内にしてください。
```

### マニュアル作成

```
あなたは業務マニュアルの作成を専門とする先輩社員です。
私は[業務名]の手順をマニュアルにまとめたいです。
対象：[誰向けのマニュアルか]
以下の作業メモをもとに、初めての人でも分かるマニュアルを作ってください。
[作業メモ]
```

## 練習ワーク

1. 自分の業務で使う「お願い文」を基本型を使って1つ書く
2. ChatGPTに試してみる
3. 回答を見て、足りない情報を追加して再度試す

> ⚠️ **注意：** 個人情報・顧客情報・機密情報は入力しないでください。
